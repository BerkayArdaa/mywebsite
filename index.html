2D Adventure Game with Unity

During my internship at Teracity Software Technologies, I developed a complete 2D action-adventure game using Unity and C#. This project included core gameplay mechanics such as player movement, combat, inventory management, wave-based enemy spawning, and boss battles.

One of the highlights was integrating a dynamic weapon system that allows players to switch between sword and bow. The bow features projectile mechanics where arrows follow the mouse direction using Rigidbody2D physics. I also designed an interactive inventory system with slot-based item management, item pickups, drops, and use-case logic (e.g., potions that provide power-ups).

To enhance game flow, I added a scoring system, visual UI components using TextMesh Pro, and enemy AI logic including enemy movement, ranged attacks, and player tracking. Special care was taken to optimize performance and avoid memory issues by destroying unnecessary objects after a certain time. Kormos ‚Äì Supernatural Detective Adventure

Kormos is a 3D adventure game where players control a supernatural detective equipped with unique powers to solve a dark mystery. Built in Unity, the game combines action, puzzle-solving, and tactical combat across a mysterious map filled with dangerous enemies.

The detective character has special abilities, including Clone Summoning, where clones appear around a selected enemy and attack, and Time Manipulation, which slows time to allow more strategic attacks. The UI displays critical information such as health and ammo, and a dynamic crosshair system helps with ranged targeting.

Enemies include wandering Rats, which chase and attack the player in close range, and Priests, who patrol and launch magic projectiles from afar. At the end of the first level, players face the Rat King, a powerful boss that summons rats while dealing heavy damage.

All characters, animations, and visual assets were integrated using Unity's prefab system and the Unity Asset Store. This project emphasized real-time combat mechanics, animation triggers, enemy AI, and user-friendly UI design.
